Biography

Tentang Saya

Halo, Saya Andre.

Saya adalah seorang Web Developer asal Indonesia yang bersemangat dalam menggabungkan fungsionalitas backend dan seni visual frontend. Melalui akun GitHub @NamekianPiccolo, saya aktif bereksperimen dengan berbagai library dan membangun aplikasi web yang bermanfaat untuk kehidupan sehari-hari, seperti sistem pengelolaan laundry dan marketplace sederhana.

Saya percaya bahwa sebuah website harus cepat diakses, mudah dibaca (readability), dan memiliki sentuhan estetika premium tanpa mengurangi performa.
